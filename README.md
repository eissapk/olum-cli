# olum-cli
> A CLI tool for olumjs 

[npm](https://www.npmjs.com/package/olum-cli)

### Getting Started
#### Installation
```bash
npm i -g olum-cli
```

#### Create project
```bash
olum create my-project
```

#### Navigate to project
```bash
cd my-project
```

#### Build in dev mode
```bash
npm run dev
```

#### Build for deployment
```bash
npm run build
```
